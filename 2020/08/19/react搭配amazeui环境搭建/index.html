Amazeui组件
Web组件基于 React.js 封装的 Web 组件，在 React 中，一切皆组件，无论是展示性的，还是带有交互功能的，都是组件；所有组件都是使用 JS 编写、渲染。

Amaze UI React 组件分为一下几个类型：

  • 布局相关：用于布局的组件，如网格等；
  • HTML 元素：文本域、按钮、表格等 HTML 元素增强封装；
  • 导航相关：用于导航的组件；
  • 常用相关：其他常用的组件，如图标等；
  • 交互组件：带交互功能的组件，如模态窗口等；
  • 移动组件：针对移动端设计的组件。
